Plant Complex Map Files

  • Protein Complexes
    • Description: Annotated complexes generated from walktrap diffusion clustering of FDR <10% interactions (CF-MS score cutoff 0.504)
  • Protein Interaction Network with probability scores
    • Description: Co-complex protein pairs that are observed in the protein complex map with the corresponding CF-MS score.
    • Note: CF-MS score > 0.5 = 10% FDR, > 0.2 = 50% FDR
    • Format: OrthogroupID1 [tab] OrthogroupID2 [tab] Score

Additional database input files

  • Orthogroup Annotations
    • Description: Annotations for all orthogroups
  • Translation Table
    • Mapping of eggNOG virNOG OrthogroupID to ProteinIDs
    • Generated with EggNOG mapper against EggNOG4.0 orthogroups
    • Format: OrthogroupID1 [tab] Species code [tab] ProteinID

Test and training data

  • Train Complexes
    • Description: List of training complexes used in protein complex discovery pipeline
    • Format: geneid geneid geneid ... (one complex per line)
  • Test Complexes
    • Description: List of test complexes used in protein complex discovery pipeline
    • Format: geneid geneid geneid ... (one complex per line)
  • Train Positive PPIs
    • Description: List of train postive ppis used in protein complex discovery pipeline
    • Format: geneid geneid
  • Train Negative PPIs
    • Description: List of train negative ppis used in protein complex discovery pipeline
    • Format: geneid geneid
  • Test Positive PPIs
    • Description: List of test positive ppis used in protein complex discovery pipeline
    • Format: geneid geneid
  • Test Negative PPIs
    • Description: List of test negative ppis used in protein complex discovery pipeline
    • Format: geneid geneid

Elution table

  • Full elution profile
    • Description: Normalized elution profile of all observed orthogroups
    • Format: OrthogroupID,[Fractions]
  • Tidy elution profile
    • Description: Tidy elution of all proteins
    • Format: OrthogroupID,FractionID,PSM,Normalized PSM,...

Feature Matrix

  • Feature Matrix (gz)
    • Description: Pre-feature selection features from McWhite & Papoulas et al.
    • Format: OrthogroupID,OrthogroupID,[features]
